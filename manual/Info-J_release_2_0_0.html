Ver 1.0から2.xへバージョンアップされた方へ

2013年1月1日にVer 2.0でジャイロセンサー対応版をリリースしました。
それ以前の版ではiPad本体の姿勢の把握に加速度センサーを使っていましたが、これに代わってジャイロセンサーを使うことにより、操作が格段に簡単で確実になりました。 "Display"ボタンを押してiPadを起こした時の姿勢がジャイロセンサーの基準となり、それからの傾き角度がリアルタイムにサインの拡大表示の位置に反映されます。
もはや事前に数回スウィングして1往復の平均時間（周期）を計る同期作業が不要になり、スウィングする速さを変えてもサイン表示が乱れることはなくなりました。 そのため、iPad 2以降のジャイロセンサー搭載モデルでは、ジャイロセンサー使用の方式の選択（バージョンアップ時の初期設定）をお勧めします。
ただし、以下の事項には注意して下さい。

  • 30秒以上スウィングしていると、ジャイロセンサーの基準値がiPadを起こした時の姿勢から徐々にずれることにより、サインの表示が傾く場合があります。この場合は、一旦iPadを平らにしてスウィング表示を止め、再度"Display"ボタンをタップしてスウィングし直して下さい。
  • 基準となる起こした時のiPadの姿勢には、画面が東西南北のどちらに向いているかも影響します。扇型スタイルでスウィングしているときに、この画面の向きを変えるとサイン表示がスウィングにうまく同期しなくなるので、変えないようにして下さい。または、iPadを起こすときに必ず相手の正面に画面を向けるようにしてください。

どちらの方式になっているかは設定画面（起動時のデザイン画面の右下「設定」の歯車マークから）で確認して下さい。

なお初代iPadのみジャイロセンサーが搭載されていないので、従来どおり加速度センサーを使った方式でご使用下さい。
